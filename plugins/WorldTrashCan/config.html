# 2.世界垃圾桶（又称家园垃圾桶）：玩家创建的，独立于世界的垃圾桶，每次清理垃圾时会将该世界的垃圾放到该世界的世界垃圾桶里
# 3.公共垃圾桶（又称全局垃圾桶）：所有世界互通的全局垃圾桶，当玩家的个人垃圾桶或任意一个世界垃圾桶塞不下时、会转移到公共垃圾桶里
