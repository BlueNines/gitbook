# 玩家公会

::: tip 介绍
玩家公会插件是一款功能强大的公会管理插件，支持多种公会管理功能。
:::

## 主要功能
- 创建公会
- 公会管理
- 公会等级
- 公会商店
- 公会任务
- 公会战争
- 公会活动

## 使用说明
1. [安装教程](#安装教程)
2. [权限说明](#权限说明)
3. [指令说明](command.md)
4. [配置说明](config.md)

## 安装教程
1. 下载插件
2. 放入plugins文件夹
3. 重启服务器
4. 修改配置文件
5. 再次重启服务器

## 权限说明
| 权限节点 | 说明 | 默认 |
|---------|------|------|
| playerguild.admin | 管理员权限 | OP |
| playerguild.use | 玩家使用权限 | true |

## 变量说明
::: tip 提示
需要安装 PlaceholderAPI 才能使用变量
:::

| 变量名 | 说明 |
|--------|------|
| %playerguild_name% | 公会名称 |
| %playerguild_level% | 公会等级 |
| %playerguild_money% | 公会资金 |
