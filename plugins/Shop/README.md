# 商店

## 插件介绍
商店插件是一款功能完善的服务器商店系统，支持多种商品类型、自定义界面和动态价格系统。

## 插件功能
- GUI商店界面
- 自定义商品
- 动态价格系统
- 限时商品
- 特殊商品
- 购买限制
- 交易记录
- 全球市场

## 插件命令
- [点击查看命令说明](command.md)

## 插件权限
| 权限节点 | 说明 | 默认 |
|---------|------|------|
| shop.admin | 管理员权限 | OP |
| shop.use | 玩家使用权限 | true |
| shop.sell | 出售物品权限 | true |

## 插件变量
| 变量名 | 说明 |
|--------|------|
| %shop_balance% | 玩家余额 |
| %shop_sales% | 销售总额 |
| %shop_purchases% | 购买次数 |
